# Conversor de Temperatura

Aplicação web para conversão precisa de valores entre as escalas termométricas Celsius, Fahrenheit e Kelvin. Desenvolvida com HTML5, CSS3 e JavaScript puro, a ferramenta apresenta uma interface visual limpa, responsiva e de operação intuitiva, permitindo que o usuário obtenha resultados imediatos com apenas alguns cliques.

🔗 Acesse o projeto: [https://ruizxc123.github.io/Conversor_De_Temperatura/](https://ruizxc123.github.io/Conversor_De_Temperatura/)

---

## Funcionalidades

- **Conversão bidirecional** entre Celsius (°C), Fahrenheit (°F) e Kelvin (K).  
- **Interface simplificada**: campos de entrada e seletores dispostos de forma clara e acessível.  
- **Resultado instantâneo**: a conversão é realizada em tempo real ao pressionar o botão ou tecla Enter.  
- **Design responsivo e moderno**: layout adaptável a diferentes tamanhos de tela, com efeitos visuais sutis que melhoram a experiência do usuário.

## Objetivo

Este projeto foi concebido como exercício prático para consolidação de fundamentos do desenvolvimento web front-end. Durante sua construção, foram aplicados conceitos como:

- Manipulação de elementos do DOM e eventos do usuário.
- Implementação de fórmulas matemáticas de conversão de temperatura.
- Criação de interfaces responsivas com CSS moderno (Flexbox, Grid, variáveis).
- Organização de código e boas práticas de versionamento.

A aplicação resultante é funcional, de fácil manutenção e serve como exemplo de um componente interativo que pode ser integrado a projetos maiores ou utilizado como ferramenta de estudos.
